Project Overview

A simple and elegant web app that allows users to save, manage, and visit their favorite websites easily. The project focuses on DOM manipulation, form validation, and dynamic data rendering — all done with pure JavaScript, HTML, and CSS.

✅ Tech stack: HTML5, CSS3, JavaScript (ES6+)
✅ Key features: add, display, and delete bookmarks; input validation for proper URLs; dynamic rendering without page reload; and clean responsive design.

This project demonstrates my ability to build functional web applications using vanilla JavaScript — focusing on clean UI, usability, and efficient front-end logic.
