Project Overview

This project is a full-featured e-commerce web application developed during the NTI training program. It includes both a frontend built with Angular and a backend REST API built with Node.js, Express and MongoDB. Key features include user authentication (JWT), role-based access (user/admin), product and category management, shopping cart, orders, and an admin dashboard.

✅ Tech stack: Angular 20, Bootstrap 5, RxJS on the frontend; Node.js, Express, MongoDB/Mongoose on the backend.
✅ Highlights: secure login & register, product filtering/search, persistent cart, order history, admin CRUD operations, image uploads, and caching for performance.

I designed this project to demonstrate both frontend UI/UX skills and backend architecture—showing that I can build end-to-end web applications ready for deployment.
